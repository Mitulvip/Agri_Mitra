# 🌾 AI Farmer Advisor

An AI-powered web application that helps farmers make informed agricultural decisions using machine learning and data-driven insights.  
The platform provides intelligent recommendations for crops, fertilizers, and crop disease detection, enabling farmers to improve productivity and adopt smarter farming practices.

---

## 🚀 Features

### 🌱 Crop Recommendation
Suggests the most suitable crops based on:
- Soil nutrients (Nitrogen, Phosphorus, Potassium)
- Temperature
- Humidity
- Rainfall
- Soil conditions

### 🧪 Fertilizer Recommendation
Recommends optimal fertilizers depending on:
- Soil nutrient composition
- Selected crop
- Nutrient deficiency analysis

### 🌿 Crop Disease Detection
Uses deep learning models to:
- Analyze plant images
- Detect possible diseases
- Suggest preventive measures

### 📊 Data-driven Farming
Provides insights to help farmers:
- Improve crop yield
- Reduce fertilizer misuse
- Make climate-aware farming decisions
